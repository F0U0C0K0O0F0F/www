HTML language

The HTML language is essential knowledge that you need to have in order to create websites. It's pretty simple and all websites use it. Advanced websites use a few more languages along with HTML.

Editor

It's not a good idea to use a regular Notepad to write HTML code since they lack a lot of necessary features like code highlighting and UTF encoding support.

Notepad++

As you are a beginner and will only be making basic webpages, for now, I recommend using Notepad++ (an advanced text editor).

HTML page

An HTML page is a very simple thing to create. Essentially, all it is, is a text document.

Tags

HTML consists of marks (aka tags). HTML's main purpose is to give meaning to specific parts of the document. At first, HTML was also used for graphic styling which resulted in confusing websites and has been limited to editing web content.

Doctype

HTML files have a specific structure that they have to follow. The first thing we have to add is the doctype at the very beginning of the file.

HTML

The entire document is placed inside the HTML element. This element is then split up into 2 more: head and body.
