Základy
Komentáre, vstup/výstup, literaly
Komentáre a docstringy

print(obj, sep=' ', end='\\n', file=sys.stdout)
Vypíše hodnoty do konzoly. Parametre pre oddelovač, ukončenie a cieľový súbor.

input(prompt)
Načíta riadok textu od používateľa (vracia str).

None
Špeciálna hodnota indikujúca absenciu hodnoty.

Konverzie typov
int(x), float(x), str(x), bool(x), list(x), tuple(x), set(x), dict(x)
Explicitné prevody medzi typmi; pozor na ValueError pri neplatných vstupoch.

Operátory

Aritmetické operátory
+, -, *, /, //, %, **
Sčítanie, odčítanie, násobenie, float delenie, celočíselné delenie, zvyšok, mocnina.
5 + 3\n7 // 3 # 2\n2 ** 3 # 8
Porovnávacie operátory
==, !=, <, >, <=, >=
Porovnania vracajú bool.

Logické operátory
and, or, not
Logické operácie; 'and' a 'or' sú krátkodobé (short-circuit) a vracajú pôvodné hodnoty.
a and b\nnot x
Bitové operátory
&, |, ^, ~, <<, >>
Bitové operácie nad celými číslami.

Priraďovanie / augmented
=, +=, -=, *=, /=, //=, %=, **=
Zkrátené priraďovanie pre aktualizáciu hodnoty v mieste.
x += 1\ns *= 2
Členstvo / Identity
in, not in, is, is not
Test členstva v kolekcii; 'is' porovnáva identitu objektu.

Riadiace štruktúry
if/for/while/try/with
If / Elif / Else
if podmienka:\n ...\nelif iná:\n ...\nelse:\n ...
Vetvenie programu podľa podmienok. Odsadenie je syntaktický blok.
if x > 0:\n print("kladné")\nelse:\n print("nekladné")
výraz_if_true if podmienka else výraz_if_false
Krátky zápis pre podmienku vo výraze.
res = "OK" if ok else "ERR"
For / While / else v cykloch
for item in iterable:\n ...\nelse:\n ...
For iteruje cez iterable; voliteľný blok else sa vykoná ak sa cyklus nedokončil cez break.
for i in range(5):\n print(i)\nelse:\n print("done")
while podmienka:\n ...\nelse:\n ...
While vykonáva opakovane kým je podmienka True; else podobne ako pri for.
i = 0\nwhile i < 3:\n i += 1\n if i == 2: continue\n print(i)
With / context managers
with expression as var:\n ...
Zabezpečí korektne zatvorenie/úpravu zdrojov (súbory, zámky, ...).
with open('a.txt','w') as f:\n f.write('ahoj')
Try / Except / Finally / Raise
try:\n ...\nexcept SomeError as e:\n ...\nfinally:\n ...
Ošetrenie chýb a povinné čisté ukončenie blokov.
try:\n x = int(input())\nexcept ValueError:\n print('zlé číslo')\nfinally:\n print('koniec')
Funkcie
def, lambda, args/kwargs, anotácie, partial
def / return / docstring
def meno(parametre):\n '''doc'''\n return hodnota
Definícia funkcie, návrat hodnoty cez return; bez return vracia None.

Pozície / Kľúčové / Default / Varargs
def f(a, b=2, *args, c, **kwargs): ...
*args - n-tica pozičných; **kwargs - dict kľúčových. Podpora default hodnôt a keyword-only parametrov.
def f(a, b=2, *args, c=3, **kwargs):\n print(a,b,args,c,kwargs)
Lambda / map / filter / reduce
lambda x: x*2\nlist(map(lambda x: x*2, [1,2]))
Krátke anonymné funkcie, často používané s map/filter, ale list comprehensions sú preferované.
squares = [x*x for x in range(5)]
Dekorátory (decorators)
@decorator\ndef f(...): ...
Funkcie, ktoré obalia iné funkcie. Používajú sa pre logging, caching, autorizáciu...
def log(fn):\n def wrapper(*a, **k):\n print('call')\n return fn(*a, **k)\n return wrapper\n\n@log\ndef greet():\n print('hi')

Rôzne spôsoby importu. Preferuj explicitné importy (from modul import name) pre čitateľnosť.

Kolekcie
list, tuple, dict, set, comprehension
List / Tuple
l = [1,2]\nl.append(x)\nl.pop()\nl[1:3]
Listy sú mutable, podporujú metódy append/extend/pop/insert, slicing vracia nový list.
l = [1,2,3]\nl.append(4)\nprint(l)
t = (1,2)
Tuples sú immutable (nemenné), vhodné pre heterogénne kolekcie a kľúče do dict.

Dict

Slovníky mapujú kľúče na hodnoty; používa sa get pre bezpečný prístup, collections.defaultdict pre default hodnoty.

Set

Neusporiadaná kolekcia unikátnych prvkov; podpora množinových operácií.

Comprehensions

Kratké a výkonné spôsoby tvorby kolekcií; generator expressions sú lazy.

Iterátory

Iterator je objekt s metodami __iter__ a __next__; next() číta ďalší prvok alebo vyhodí StopIteration.

Generátory (yield)
def gen():\n yield 1\n yield 2
Generátorové funkcie produkujú hodnoty postupne (lazy evaluation), vhodné pri práci s veľkými dátami.

Threading

Vlákna sú užitočné pre I/O bound úlohy; pri CPU-bound úlohách GIL obmedzuje paralelizmus.

Spúšťa viac procesov — obchádza GIL, vhodné pre CPU-bound úlohy.

Asynchrónny model vhodný pre veľa súbežných I/O; používa event loop a await pre awaitables.
