Python — rýchla tabuľka príkazov a konštrukcií

Logicky usporiadané vysvetlenia pre začiatočníkov: syntax, popis a krátky príklad.

Príkaz / Konštrukcia Syntax Popis Krátky príklad
Základy
Komentár # komentár Jednoradkový komentár — ignorovaný interpreterom; vysvetlenie kódu pre čitateľa. # Toto je komentár
Výpis print(obj) Vypíše text alebo hodnotu na štandardný výstup (konzola). print("Ahoj, svet!")
Vstup input(prompt) Prečíta text od používateľa ako reťazec. name = input("Meno: ")
Premenné a dátové typy
Priradenie x = 5 Vytvorenie/aktualizácia premennej. Python je dynamicky typovaný. x = 10
y = "text"
Typy int, float, str, bool, list, tuple, dict, set Základné dátové typy. Používaj vhodný typ podľa potreby. a = 3 # int
b = 3.5 # float
c = "ahoj" # str
Konverzia typov int(x), float(x), str(x), list(x) Prevody medzi typmi; pozor na chyby pri neplatnom vstupe. n = int("5") # 5
Operátory
Aritmetické +, -, *, /, //, %, ** Sčítanie, odčítanie, násobenie, delenie (float), celočíselné delenie, zvyšok, mocnina. 5 ** 2 # 25
7 // 3 # 2
Porovnávacie ==, !=, <, >, <=, >= Vrátia bool výsledok (True/False) pre porovnania hodnôt. 3 < 5 # True
Logické and, or, not Logické operácie používané v podmienkach. a and b
not cond
Riadiace štruktúry
If / elif / else if podmienka:
  ...
elif iná:
  ...
else:		 Vyhodnocuje vetvenie programu podľa podmienok. if x > 0:\n print("kladné")\nelse:\n print("nie kladné")
For for item in iterable: Iteruje cez sekvenciu (list, tuple, string, range, ...). for i in range(3):\n print(i)
While while podmienka: Cyklicky vykonáva blok kódu kým je podmienka True. i = 0\nwhile i < 3:\n i += 1
Break / Continue / Pass break, continue, pass break ukončí slučku; continue preskočí iteráciu; pass nič nerobí (placeholder). for x in t:\n if x==0: break\n if x<0: continue
Funkcie
Definícia funkcie def name(params): Vytvorenie znovupoužiteľného bloku kódu. Funkcia môže vracať hodnotu cez return. def add(a,b):\n return a+b\nprint(add(2,3))
Lambda lambda x: x*2 Anonymná (krátka) funkcia, vhodná pre krátke jednorazové použitia. d = list(map(lambda x: x*2, [1,2,3]))
Moduly a balíky
Import import modul
from modul import name		 Načítanie knižníc (modulov). Odporúča sa používať explicitné importy. import math\nprint(math.pi)
Vlastné moduly # súbor utils.py
def f(): ...		 Umiestni kód do samostatného súboru a importuj ho v hlavnom skripte. # v main.py\nfrom utils import f\nf()
Výnimky (chyby)
Try / Except / Finally try: ...\nexcept ErrorType as e:\n ...\nfinally: Ošetrenie chýb, aby program neskončil neočakávane. try:\n x = int(input())\nexcept ValueError:\n print("Neplatné číslo")
Raise raise ValueError("msg") Vynúti výnimku ručne, používa sa pri kontrole vstupov a chybách. if x<0: raise ValueError("x musí byť >=0")
Práca so súbormi
Otvorenie súboru with open("file","r") as f: Bezpečný spôsob práce so súbormi — automaticky zatvorí súbor. with open("a.txt","w") as f:\n f.write("ahoj")
Triedy a OOP
Definícia triedy class Name: Vytváranie vlastných typov (objektov) s atribútmi a metódami. class Person:\n def __init__(self,name):\n self.name = name
Inštancia obj = ClassName() Vytvorenie konkrétneho objektu z triedy. p = Person("Eva")\nprint(p.name)
Užitečné funkcie / tipy
range range(start, stop, step) Generuje číselnú sekvenciu — efektívne pre cykly. for i in range(5): print(i)
len len(obj) Vráti dĺžku sekvencie alebo veľkosť kolekcie. len([1,2,3]) # 3
enumerate / zip enumerate(iterable), zip(a,b) enumerate pridá index; zip spája viac sekvencií dohromady. for i,v in enumerate(items):\n print(i,v)
List comprehensions [x*2 for x in seq if cond] Krátky a prehľadný spôsob vytvárania zoznamov. squares = [x*x for x in range(5)]
Docstring """popis funkcie""" Popis funkcie/triedy — používa sa pre dokumentáciu (help()). def f():\n \"\"\"Vysvetlenie co robi funkcia\"\"\"\n pass
